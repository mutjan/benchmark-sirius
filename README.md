# シリウスの心臓 Benchmark 展示页

## 使用方法

### 1. 启动本地服务器

```bash
cd benchmark-sirius
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

### 2. 添加新模型测试结果

编辑 `data/scores.json`，在 `results` 数组中添加一条记录：

```json
{
  "id": "gpt-4o-2024-11-20",
  "model": "GPT-4o",
  "model_version": "gpt-4o-2024-11-20",
  "provider": "OpenAI",
  "api_endpoint": "api.openai.com",
  "test_date": "2026-03-21",
  "notes": "",
  "scores": {
    "Q1": 1,
    "Q2": 1,
    "Q3": 1,
    "Q4": 1,
    "Q5": 2,
    "Q6": 2,
    "Q7": 2,
    "Q8": 1,
    "Q9": 1
  }
}
```

刷新页面即可看到新模型出现在排行榜和雷达图中。

### 评分规则

| 题目 | 满分 | 维度 |
|------|------|------|
| Q1 符号识别 | 1 | 符号识别 |
| Q2 解码结果 | 1 | 符号识别 |
| Q3 元语言密钥 | 2 | 元语言密钥 |
| Q4 词语递进 | 2 | 语言深度 |
| Q5 副歌演变 | 3 | 语言深度 |
| Q6 意象关联 | 3 | 天文知识 |
| Q7 引力词义 | 3 | 天文知识 |
| Q8 天文背景 | 3 | 天文知识 |
| Q9 发布时间意义 | 5 | 时间推理 |

分值可以是小数（如 1.5、0.5）。
